# React-Node-MongoDB

Follow the below mentioned steps :

1> Make sure you have Node.JS and, MongoDb Compass Community installed in your system.

2> Clone repository in your local system.

3> Open "backend" folder in a command prompt and run command "npm install" to install all the required modules.

4> Run "npm start" command which will start your application on 3000 port and will be pointing to your local Mongo DB.

5> Open "frontend" folder in a command prompt and run command "npm install" to install all the required modules.

6> This application will start on port 3001 and will be pointing on the api's which are running on 3000 port.
